Проект социальной сети с возможностью:

- регистрации пользователей с контролем аутентификации через WEB и API(Token)
- создание и редактирование профиля с расширенными полями (день рождения, фото и т.д.)
- создание и редактирование постов
- подписка на любимых авторов (с возможностью отписаться)
- создание и редактирование комментариев к постам


Дополнительный функционал:

- в проекте реализованы Django tests (документация) для проверки фунционала кода
- подключение Telegram bot для получения информации о последнем посте (опрос API раз в 20 сек) и получение конкретного поста по запросу вида "/post 12"


REST API проекта:

При использовании Swagger для методов POST/PATCH/PUT/DELETE необходимо авторизоваться введя в поле авторизации значение Token через пробел ПРИМЕР (Token rfd543vfc....gfddc)
- Просмотр API спецификации через api/v1/swagger/
- Просмотр JSON спецификации через api/v1/swagger.json
- Просмотр YAML спецификации через api/v1/redoc/

GraphQL - API проекта (только метод Query):

- Просмотр API через api/v1/graphql/
